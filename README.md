# 🎒 초등 맞춤형 조사학습 앱 (단일 HTML 기반)

본 프로젝트는 별도의 서버(백엔드) 없이 단일 HTML 파일로 구동되는 **초등학생 맞춤형 안전 검색 및 탐구 기록 웹앱**입니다. 교사가 사전에 설정한 학습 키워드와 안전한 웹사이트(화이트리스트) 내에서만 검색이 이루어지며, 학생들은 3단 변신 탐구 기록장(공책, 타자, 펜슬)을 활용해 학습 결과를 이미지로 저장할 수 있습니다.

## ✨ 주요 기능
* **서버리스 아키텍처:** 데이터베이스 없이 구글 스프레드시트와 `localStorage`만으로 작동하여 유지 비용이 0원입니다.
* **역할 분리 접속:** 교사용 설정 모드와 학생용 실습 모드가 완벽히 분리되어 있습니다.
* **QR 코드 자동 입장:** 복잡한 주소 입력 없이 1초 만에 학생들을 교사 설정 화면으로 초대합니다. (최초 1회 스캔 후 자동 로그인 지원)
* **안전 검색 연동:** 구글 맞춤검색(CSE)을 활용하여 교사가 허용한 유익한 사이트만 검색 결과에 노출합니다.
* **3단 변신 탐구 기록장:** 실제 배움 공책, 키보드 타이핑, 스마트펜 드로잉 등 학생의 기기와 성향에 맞게 기록 방식을 선택할 수 있습니다.
* **튜토리얼 모드:** 처음 접속하는 사용자를 위한 가상의 더미 데이터 기반 체험(안내 가이드 포함)을 제공합니다.

## 👩‍🏫 교사 사용 설명서 (초기 세팅 방법)

이 앱을 학생들에게 배포하기 위해서는 **구글 스프레드시트 주소**와 **검색엔진 ID** 두 가지가 필요합니다. 

### Step 1. 구글 스프레드시트(데이터베이스) 준비하기
학생들에게 보여줄 단원별 추천 키워드를 엑셀처럼 표로 정리합니다.

1. 새 구글 스프레드시트를 생성합니다.
2. **1행(첫 줄)**에 정확히 아래와 같은 이름으로 열 제목을 적어주세요. (띄어쓰기 없이 작성)
   > `학년`, `과목`, `단원`, `차시`, `기본키워드`, `융합키워드`
3. 2행부터 실제 수업 데이터를 입력합니다. 키워드가 여러 개일 경우 **쉼표(,)**로 구분합니다.
   
   **[작성 예시]**
   | 학년 | 과목 | 단원 | 차시 | 기본키워드 | 융합키워드 |
   | :--- | :--- | :--- | :--- | :--- | :--- |
   | 3학년 1학기 | 과학 | 4단원. 화산과 지진 | 1차시. 화산 분출물 | 화산재, 용암, 화산암 | 폼페이 화산 폭발 |
   | 6학년 2학기 | 사회 | 1단원. 국토와 우리 생활 | 1차시. 국토의 위치 | 영토, 영해, 영공 | 독도의 중요성 |

4. 작성 후 우측 상단 **[공유]** 버튼을 누르고, 일반 액세스 권한을 **'링크가 있는 모든 사용자(뷰어)'**로 변경한 뒤 링크를 복사합니다.

### Step 2. 구글 맞춤검색(CSE) 엔진 ID 발급 및 화이트리스트 등록
학생들이 유튜브, 게임, 유해 사이트로 빠지지 않고 학습 관련 사이트만 검색하도록 설정합니다.

1. [구글 맞춤검색 고객센터](https://cse.google.com/cse/all)에 접속하여 새 검색엔진을 만듭니다.
2. **'검색할 특정 사이트'** 항목에 학생들의 접속을 허용할 유익한 사이트(화이트리스트)를 추가합니다.

   **💡 초등 현장 추천 화이트리스트 사이트 예시:**
   * `*.go.kr` (정부 및 관공서 공식 포털 전체 허용)
   * `*.es.kr` (전국 초등학교 홈페이지 전체 허용)
   * `terms.naver.com` (네이버 지식백과)
   * `study.edunet.net` (에듀넷 티클리어 학생 공간)
   * `kids.dongascience.com` (어린이 과학동아)
   * `nationalgeographic.co.kr` (내셔널지오그래픽 한국)

3. 생성 후 **[제어판] - [기본] - [검색엔진 ID]**에 있는 고유 ID 문자열을 복사합니다.
4. (선택) 제어판의 '안전 검색' 설정을 '엄격하게 적용'으로 켭니다.

### Step 3. 앱에 설정 입력 및 배포
1. 배포된 웹앱 첫 화면에서 **[👩‍🏫 선생님]**을 클릭합니다.
2. 설정 창에 복사해 둔 '구글 시트 주소'와 '검색엔진 ID'를 붙여넣습니다.
3. **[🚀 학생 배포용 링크 생성]** 버튼을 누릅니다.
4. 화면에 나타난 링크를 학급 커뮤니티(클래스팅, e학습터 등)에 공유하거나, 나타난 **QR 코드**를 교실 TV에 띄워 학생들이 태블릿으로 찍고 들어오게 합니다.

## 👦👧 학생 사용 설명서

1. 교실 TV에 띄워진 QR 코드를 태블릿 카메라로 스캔하거나, 선생님이 올려주신 링크를 클릭합니다.
2. **[학생 모드]**를 선택하여 접속합니다. (최초 1회만 스캔하면 기기에 선생님 교실이 기억됩니다.)
3. 드롭다운 메뉴에서 학년/과목/단원/차시를 선택합니다.
4. 아래에 나타나는 추천 해시태그(`#`)를 누르거나 직접 검색창에 입력하여 궁금한 내용을 탐구합니다.
5. 오른쪽 **'나만의 탐구 기록장'**의 3가지 모드(공책, 타자, 펜슬) 중 하나를 골라 알게 된 점을 정성껏 기록합니다.
6. 기록이 끝나면 하단의 **[탐구 완료! 이미지로 저장하기]**를 눌러 사진첩에 저장하고 과제로 제출합니다.

### 🛠 기술 스택
* **HTML5 / CSS3** (Tailwind CSS CDN 활용)
* **Vanilla JavaScript** (ES6+)
* **html2canvas** (탐구 기록장 이미지 캡처/다운로드)
* **PapaParse** (구글 스프레드시트 CSV 파싱)
* **qrcode.js** (학생용 QR 코드 동적 생성)
* **jsQR** (웹캠 기반 QR 코드 스캔)
